# discuss

議論フローを開始するコマンドです。

## 実行内容

1. `prompts/12-ideas.md`を読み込み、`[議論待ち]`のアイデアを確認
2. 議論対象のアイデアを特定
3. 議論モードに入ることを明確に示す
4. 議論の結果に基づいてステータスを更新する準備をする

## 使用方法

チャットで `/discuss` と入力すると、議論フローに入ります。

## 議論時の注意事項

- `[議論待ち]`のアイデアを確認
- 議論の結果（採用/却下）を記録
- ステータスを `[議論中]` → `[議論完了]` に更新
- 議論完了後は議事録（`11-meeting-notes.md`）に移行

## 次のステップ

1. 議論が完了したら、議事録を`prompts/11-meeting-notes.md`に記録します
2. 議事録を記録したら、タスクを`prompts/14-tasks.md`に切り出します
3. タスクが作成されたら、`/design`コマンドで設計フローに入ります

詳細は `prompts/00-ai-workflow.md` を参照してください。
