# idea

アイデア記録フローを開始するコマンドです。

## 実行内容

1. `date +%Y-%m-%d`コマンドを実行して現在の日付を取得する
2. `prompts/12-ideas.md`を読み込み、既存のアイデアを確認
3. アイデア記録モードに入ることを明確に示す
4. ユーザーからのアイデアを記録する準備をする

## 使用方法

チャットで `/idea` と入力すると、アイデア記録フローに入ります。

## アイデア記録時の注意事項

- **日付は必ず`date +%Y-%m-%d`コマンドで確認してから記録する**（最新のアイデアの日付を参照しない）
- 既存のアイデアと重複していないか確認
- テンプレートに従って記録（`prompts/12-ideas.md`参照）
- ステータスは `[議論待ち]` で記録
- 日付、内容、提案者、優先度を記載

## 次のステップ

アイデアが整理できたら、`/discuss` コマンドで議論フローに入ります。

詳細は `prompts/00-ai-workflow.md` を参照してください。
