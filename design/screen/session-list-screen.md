# セッション一覧画面

## 概要

セッション一覧を表示し、セッションの作成・編集・削除を行う画面です。

## 画面構成

### レイアウト

- ヘッダー: 画面タイトル「セッション一覧」
- メインコンテンツ:
  - 新規作成ボタン
  - セッション一覧テーブル
  - 編集・削除ボタン（各行）

## 画面項目

### ヘッダー

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| 画面タイトル | テキスト | 「セッション一覧」 |

### 新規作成ボタン

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| 新規作成ボタン | ボタン | セッション作成画面に遷移 |

### セッション一覧テーブル

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| 日付 | 日付 | セッションの日付（YYYY-MM-DD形式） |
| セッション名 | テキスト | セッションの名前（未設定の場合は「無題」と表示） |
| 参加者 | テキスト | 参加者名をカンマ区切りで表示 |
| 参加者数 | 数値 | 参加者の人数 |
| 半荘数 | 数値 | そのセッションに紐づく半荘の数 |
| 編集ボタン | ボタン | セッション編集画面に遷移 |
| 削除ボタン | ボタン | セッションを削除（確認ダイアログ表示） |
| 詳細ボタン | ボタン | セッション詳細画面に遷移 |

## 画面遷移

### 遷移元

- なし（初期画面、またはナビゲーションメニューから）

### 遷移先

- セッション作成画面（新規作成ボタンクリック時）
- セッション編集画面（編集ボタンクリック時）
- セッション詳細画面（詳細ボタンクリック時、または行クリック時）

## 機能

### セッション一覧取得

- 画面表示時に`GET /api/sessions`を呼び出し、セッション一覧を取得
- 取得したデータをテーブルに表示
- 日付順（新しいものから）で表示
- エラー時はエラーメッセージを表示

### セッション削除

- 削除ボタンクリック時に確認ダイアログを表示
- 確認後、`DELETE /api/sessions/:id`を呼び出し、セッションを削除
- 削除成功後、一覧を再取得
- エラー時はエラーメッセージを表示

## リアルタイム更新

- セッションの作成・更新・削除後、一覧を自動更新
- 他の画面から戻ってきた場合も一覧を再取得

## UI/UX

### デザイン

- Vuetifyのコンポーネントを使用
- テーブルは`v-data-table`を使用
- ボタンは`v-btn`を使用

### エラーハンドリング

- APIエラー時は`v-alert`でエラーメッセージを表示
- ネットワークエラー時も適切にエラーメッセージを表示

### ローディング

- データ取得中はローディングインジケーターを表示

## 実装メモ

- Vue 3 Composition APIを使用
- TypeScriptで型安全性を確保
- リアルタイム更新は必要に応じて実装（現時点では手動更新）
- 参加者名は`sessionPlayers`から取得して表示
