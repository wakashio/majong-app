# About画面

## 概要

麻雀記録アプリのAbout画面です。アプリの説明、使い方、バージョン情報を表示します。

## 画面構成

### レイアウト

- メインコンテンツ:
  - アプリ説明セクション
  - 使い方セクション
  - バージョン情報セクション

## 画面項目

### アプリ説明セクション

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| アプリ名 | テキスト | 「麻雀記録アプリ」 |
| アプリ説明 | テキスト | アプリの概要説明 |
| 機能一覧 | リスト | 主な機能の一覧 |

### 使い方セクション

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| セクションタイトル | テキスト | 「使い方」 |
| 使い方説明 | テキスト | アプリの基本的な使い方の説明 |
| 手順リスト | リスト | 使用手順のリスト |

### バージョン情報セクション

| 項目名 | タイプ | 説明 |
|--------|--------|------|
| セクションタイトル | テキスト | 「バージョン情報」 |
| バージョン番号 | テキスト | アプリのバージョン番号（例: 1.0.0） |
| 更新日 | 日付 | 最終更新日 |
| ライセンス情報 | テキスト | ライセンス情報（オプション） |

## 画面遷移

### 遷移元

- ナビゲーションドロワーから「About」を選択
- フッターの「About」リンク（オプション）

### 遷移先

- なし（情報表示のみ）

## 機能

### 情報表示

- 静的な情報を表示（API呼び出しなし）
- バージョン情報は`package.json`から取得することも可能

## UI/UX

### デザイン

- Vuetifyのコンポーネントを使用
- カード形式でセクションを表示（`v-card`を使用）
- リストは`v-list`を使用
- テキストは`v-card-text`を使用

### レスポンシブデザイン

- モバイル端末でも見やすいレイアウト
- グリッドレイアウトを使用（`v-row`、`v-col`）

## 実装メモ

- Vue 3 Composition APIを使用
- TypeScriptで型安全性を確保
- 静的な情報を表示するため、API呼び出しは不要
- バージョン情報は`package.json`から取得することも可能（`import.meta.env`を使用）
