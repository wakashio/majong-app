# プロジェクト概要

## プロジェクトの目的

Salesforce 認定試験の学習を支援するクイズアプリケーションです。受験者が効率的に学習できるよう、問題の表示、解答、進捗管理、統計情報の提供を行います。

## 主要機能

### 1. 問題の表示と解答

- CSV ファイルから問題データを読み込み、順次表示
- 単一選択問題と複数選択問題の両方に対応
- リアルタイムでの解答入力と正誤判定

### 2. 複数選択問題の対応

- 正解が複数ある問題に対応
- カンマ区切りで複数の選択肢を正解として指定可能
- チェックボックス形式での解答入力

### 3. 学習進捗の保存

- LocalStorage を使用した学習進捗の永続化
- データソースごとに独立した進捗管理
- 回答履歴、正誤情報、学習時間の記録

### 4. 統計情報の表示

- 正解率、回答済み問題数、正解数の表示
- セッション時間の記録
- 日別学習記録の管理

### 5. 間違えた問題の確認

- 間違えた問題の一覧表示
- 復習が必要な問題の管理
- 習得済み問題の記録

### 6. データソースの切り替え

- 複数の問題集（データソース）に対応
- データソースごとの独立した進捗管理
- 左サイドバーからの簡単な切り替え

## ターゲットユーザー

- Salesforce 認定試験の受験者
- 効率的な学習方法を求める学習者
- 進捗を可視化したい学習者

## ビジネス要件

### 必須要件

- オフラインでも動作可能（LocalStorage 使用）
- 複数の問題集に対応
- 学習進捗の永続化
- 直感的な UI/UX

### 将来の拡張要件

- バックエンド API による進捗のクラウド同期
- ユーザー認証機能
- 学習レポートの生成
- 問題の追加・編集機能

## プロジェクトの現状

- フロントエンド: 実装済み（Vue 3 + TypeScript）
- バックエンド: 未実装（TypeScript 環境のみ構築済み）
- データ管理: LocalStorage を使用
- デプロイ: 未実施
